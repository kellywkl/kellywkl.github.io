初识angular4

发表于 2017-09-11
字数统计: | 阅读时长 ≈

一、搭建环境

angular命令行工具安装：cnpm install -g @angular/cli
安装完成后可通过：ng -v 查看其版本

二、创建项目

ng new ng4demo [ng4demo为项目名称]
cd ng4demo
cnpm install
ng serve –open
这样我们的项目就可以运行起来了

三、第三方库安装和引入

1、jquery、bootstrap安装

2、安装完成后，修改.angular-cli.json的styles和scripts

3、在src/styles.css中引入bootstrap样式

4、在src/app/app.component.tsy引入jquery

至此，jquery和bootstrap就已经引入项目中，可以尽情的使用了。

生成组件命令:ng g component header [header为组件名称]
