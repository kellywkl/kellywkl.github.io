自定义样式无效解决方法

注意：微信的页面是 https 的，出于安全考虑，浏览器会拒绝访问 http 中的资源

解决方案：

  • 改用 https
  • 使用 data-url(将cssUrl在base64转换转成base64的url,且自定义样式文件里不能有中文注释，即可)
