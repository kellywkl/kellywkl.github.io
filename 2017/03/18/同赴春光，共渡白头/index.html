同赴春光，共渡白头

发表于 2017-03-18
字数统计: 772 字 | 阅读时长 ≈ 2 分钟


夜火太过漫长，思念不知该如何安放。是随着清风飘散，还是伴着明月悄然？走在温阳如火的二月里，青涩的梧桐淡出馨香。手捧静夜里的明珠，看烟火里一场烂漫，看流年里一记清欢。

隆冬的岁月已然逝去，带走记忆里的风霜，还有那回不去的过往。屋顶透过闪耀的星斑，把心间最美丽的情话描摹成了谁的模样？玉出的眉头是夕阳沉暮的样子，多愁善感。婉约着清脆的声音，细雨朦胧之中沾湿了谁的衣裳？一场俗世年华的盛景，一颗灼灼其华的圣心，邀约着清风与明月，来一场不散的筵席。

你不来我不走，你走了我还在。愿岁月划过指尖，风雨过后，别来无恙。

春风十里路，满树桃花开。

轻拙玉足粉，怜袖几回生。

东郭闲野趣，伴竹丝绕声。( 文章阅读网：www.sanwen.net )

青衣故沾雨，冷落是闲人。

本是一种闲来无趣的姿态，无心惊扰了这俗世有我的感慨。看尽江南水乡，绿叶红花处多出几笔暗淡。愁眉动容之时，不知流落何处？是烟火灿烂之地，还是月明星稀之所？本该相守最美的季节，可怜又是辗转流离之时。看一世桃花灼灼，遇谁人一生白首？

三月的号子即将吹响，青葱的岁月悄然来临。北风过境的时候，我依旧躲在屋里。经不起流年的刻薄，只愿一生都藏在无恙的光阴里。没有风和雪的大地是温暖的，没有悲和欢的人生是寒冷的。一曲红尘高歌，一骥飞马扬鞭，是俗世也是人生。

高楼之外，天地辽阔。迎着风去思念或是想起一个人，故去的模样早已不再清晰。像二月的花簇，逐渐淡去了颜色，直至最后慢慢凋谢。是什么夺走了身边失去的那些美景，是时间的流逝，还是恨自己没有牢牢抓紧？一夜花落，一叶花开。失去的都已成为了过去，眼前能够得到的才能够拥有。桃花夭夭，灼灼其华。世有桃花，不离不弃；世有良媒，不嗔不念。

倚靠岁月的阑干,任行将就往的故事在脑海中翻滚。抚摸日下的夕阳，亲昵这一世春光。没了熠熠的过往，当下的思念何其漫长，但也足够让你爱上此时、此生、此世。留一人在心间，容一世长情。恨也罢、爱也罢，不足惜，终究遗憾。

温阳如火，桃花先漫。人世匆匆，如白驹过隙。孤独是城，锁了你，也困了我。就让我们在这孤独的字里行间彼此相望，一同看尽繁华，看尽春光。

子若有情毋须执手，且行且就，同赴春光，共渡白头！
