JS、JQuery调用iframe父窗口与子窗口元素的方法

虽说现在iframe很少用到，但用到了你还得会。

1. jquery 在iframe子页面获取父页面元素代码如下

$(“#elementId”, parent.document)

2. jquery在父页面 获取iframe子页面的元素

$(“#elementId”, document.frames(‘iframename’).document)

3. js 在iframe子页面获取父页面元素

window.parent.document.getElementById(“元素id”);

4. js 在父页面获取iframe子页面元素

window.frames[“iframe_ID”].document.getElementById(“元素id”);

5. 子类iframe内调用父类函数：

window.parent.func();
