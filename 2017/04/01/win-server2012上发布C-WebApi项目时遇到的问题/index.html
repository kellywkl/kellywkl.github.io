Win Server2012上发布C# WebApi项目时遇到的问题

发表于 2017-04-01 | 分类于 C#
字数统计: 55 字 | 阅读时长 ≈ 1 分钟

Win Server2012上发布C# WebApi项目时，报错’找不到方法:”!!0[] System.Array.Empty()”.’

原因是服务器上没装：.net framework 4.6
解决方案：去下载安装 .net framework 4.6 然后重启服务器即可。
